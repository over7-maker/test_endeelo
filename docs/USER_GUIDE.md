# دليل المستخدم (User Guide) - Fortress v3.1

بعد الانتهاء من عملية التثبيت (راجع `installation-guide.md`)، يمكنك البدء في استخدام النظام.

## التحكم في الأنفاق (Tunneling)
يمكنك إدارة الأنفاق عبر واجهة الـ API أو السكربتات المحلية:
- لتدوير الهوية: `bash scripts/vps-layer2/rotate.sh`
- للتحقق من الحالة: `bash scripts/monitoring/check-status.sh`

## مراقبة النظام
- واجهة Prometheus متوفرة على المنفذ `9090`.
- لوحات تحكم Grafana (إذا تم تثبيتها) تعرض بيانات المرور والنشاط.

## نصائح أمنية
- لا تشارك ملفات الإعدادات في مجلد `configs` مع أي شخص.
- تأكد من تحديث النظام بشكل دوري.
